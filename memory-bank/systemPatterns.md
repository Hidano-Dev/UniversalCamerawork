# システムパターン (systemPatterns.md)

## アーキテクチャ
- モジュール構成: 
  - Core API 層 (カメラ情報取得・公開インターフェース)
  - シリアライザ層 (バイナリ・CRC16 組み込み)
  - コンバータ層 (独自バイナリ ↔ プラットフォームデータ型)
  - テストツール（C#）

## デザインパターン
- シングルトン: グローバルロガー、設定管理
- ファクトリ: シリアライザ/コンバータ生成
- ストラテジー: 異なるシリアライズ形式の切り替え
- オブザーバ: データ更新時通知

## コンポーネント関係
- Core API → シリアライザ → ファイル/ネットワーク
- 受信データ → デシリアライザ → コンバータ → 各プラットフォーム実装

## その他パターン
- CRC16チェック: シリアライザ内に組み込み
- 拡張性: プラグイン式コンバータ登録機構
