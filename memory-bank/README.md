# Memory Bank ファイル一覧

このディレクトリは、本プロジェクトの知見・タスク管理・ドキュメントを集約するための場所です。各ファイルの役割は以下の通りです。

- `activeContext.md`：現在の作業フォーカスと最新の変更履歴を管理するアクティブコンテキスト
- `chatContext.md`：AIアシスタントやチームとのチャットログや議論内容を記録
- `productContext.md`：プロジェクトの背景、狙い、解決する問題など製品コンテキストを整理
- `projectbrief.md`：プロジェクト概要、ゴール、制約条件を簡潔にまとめたブリーフ
- `systemPatterns.md`：アーキテクチャや採用デザインパターン、コンポーネント関係などシステムパターンを記述
- `techContext.md`：使用技術、開発環境、依存関係や技術制約をまとめた技術コンテキスト
- `tasks.md`：P0〜P2の優先タスクリストと進捗管理
- `progress.md`：進捗状況、完了タスク、未完了タスク、既知の課題を記録

- `context.md`：（同内容を正式版として）プロジェクトコンテキストをまとめたファイル
- `project-knowledge.md`：技術的知見、設計決定、実装パターンとアンチパターンを集約
- `project-improvements.md`：改善履歴、試行錯誤の記録と教訓を記述
- `common-patterns.md`：よく使用するコマンドやテンプレートをまとめたリファレンス
- `commands/` ディレクトリ：
  - `learnings.md`：カスタムスラッシュコマンドを定義し、新知見を自動的に整理・追記する方法を説明

各ファイルは継続的に更新し、プロジェクトの知見を蓄積・共有してください。
